02 —

U.S. Chronic Disease Indicators — health data pipeline & SQL analytics

Designed a star-schema MySQL database (1 fact, 3 dimension tables) from 30,000+ raw CDC BRFSS records spanning 55 states/territories; built a full ETL pipeline with staging, NULL handling, and type casting. Wrote 5 analytical queries using JOINs, CTEs, and window aggregations to surface geographic and demographic disparities in obesity prevalence across income, age, race, and sex strata.

Chest X-ray pneumonia classification — machine learning comparison

Compared KNN, PCR, Random Forest, and CNN models on a 5,863-image pediatric dataset; applied PCA dimensionality reduction (140 components) and 5-fold cross-validation. PCR achieved the lowest validation error at ~4.06%, with a clean write-up of trade-offs across model families.
